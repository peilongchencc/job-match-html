## 📱 项目简介

这是一个纯前端的岗位推荐展示系统，模拟移动设备界面，为用户提供职位推荐、匹配度评分和详细信息查看功能。系统采用流式加载动画，提供流畅的用户体验。

## ✨ 功能特点

### 垂直时间轴视图 (main.html)
- **流式加载动画**：职位卡片逐个出现，模拟实时推荐效果
- **时间轴设计**：中央连接线 + 动态进度条，视觉效果出色
- **智能布局**：职位卡片左右交替排列，充分利用空间
- **匹配度可视化**：
  - 高分职位（≥90分）：绿色高亮
  - 中等匹配（80-89分）：蓝色标识
  - 普通匹配（<80分）：橙色提示
- **详情弹窗**：点击卡片查看职位详细信息
- **实时状态提示**：显示当前正在分析的职位

### 列表视图 (sub_in.html)
- **卡片式布局**：清晰展示职位核心信息
- **完整信息展示**：
  - 职位名称和级别
  - 薪资范围和年终奖
  - 匹配度评分
  - 典型公司列表
- **优雅动画**：卡片依次淡入，提升视觉体验
- **交互反馈**：点击卡片有缩放效果

### 通用特性
- 🎨 **现代UI设计**：采用2025年流行的毛玻璃和渐变效果
- 📱 **移动端优化**：完美模拟iPhone尺寸（390×844px）
- 🔄 **视图切换**：支持在两种视图间自由切换
- 🎯 **响应式交互**：流畅的动画和过渡效果
- 🚀 **零依赖**：纯HTML/CSS/JavaScript实现

## 🛠️ 技术栈

- **HTML5**：语义化标签，结构清晰
- **CSS3**：
  - CSS自定义属性（CSS Variables）
  - Flexbox布局
  - 动画和过渡效果
  - 毛玻璃效果（backdrop-filter）
  - 渐变背景
- **原生JavaScript**：
  - DOM操作
  - 动态渲染
  - 事件处理
  - 定时器控制

## 📄 页面说明

### main.html - 时间轴视图

**核心特性**：
- 12个职位推荐数据
- 流式生成，每个职位间隔400-900ms
- 动态进度线跟随加载进度
- 自动滚动到最新职位
- 完成后状态提示变更

**数据结构**：
```javascript
{
  title: "职位名称",
  company: "公司名称",
  salary: "薪资范围",
  score: 匹配度分数(0-100)
}
```

**视觉层次**：
1. 状态栏（时间、信号、电量）
2. 导航头部（标题、状态、列表按钮）
3. 时间轴容器（职位卡片）
4. 浮动提示（加载状态）
5. 详情弹窗（职位详情）

### sub_in.html - 列表视图

**核心特性**：
- 5个精选职位推荐
- 完整的职位信息展示
- 淡入动画效果
- 典型公司标签展示

**数据结构**：
```javascript
{
  job_title: "职位名称",
  salary_range: "薪资范围",
  salary_annual_bonus: "年终奖",
  typical_companies: ["公司1", "公司2"],
  score: 匹配度分数(0-100)
}
```

## 🎨 设计亮点

### 配色方案
- **主色调**：科技蓝 `#2563EB`
- **背景色**：浅灰 `#F8FAFC`
- **文字色**：深灰 `#1E293B` / 次要 `#64748B`
- **强调色**：
  - 薪资橙 `#F97316`
  - 高分绿 `#059669`
  - 警示橙 `#D97706`

### 动画效果
- **入场动画**：淡入 + 上移
- **交互反馈**：缩放 + 阴影变化
- **加载动画**：呼吸灯效果
- **进度线**：线性增长动画

### UX优化
- 隐藏滚动条但保留功能
- 模拟iPhone状态栏和圆角
- 毛玻璃效果增强层次感
- 点击反馈提升交互体验
- 自动滚动跟随内容加载

```css

    --primary-blue: #2563EB;   /* 主色调 */
    --bg-color: #F8FAFC;       /* 背景色 */
    --text-main: #1E293B;      /* 主文字色 */
    /* ... 更多颜色变量 */
}
```

## 🔧 技术细节

### 性能优化
- 使用 `requestAnimationFrame` 优化动画性能
- CSS `transform` 代替 `position` 进行动画
- 事件委托减少监听器数量
- 隐藏滚动条减少重绘

### 兼容性处理
- CSS前缀支持旧版浏览器
- 渐进增强设计
- 优雅降级策略

### 响应式设计
- 固定手机尺寸容器
- 百分比布局适配内容
- Flexbox弹性布局
